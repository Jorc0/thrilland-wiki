# Cómo Maximizar tus Ingresos con Jobs

Si quieres hacer fortuna en Thrilland, el sistema de trabajos (Jobs) es una de las mejores formas de conseguirlo. Esta guía te enseñará estrategias específicas para cada trabajo y cómo combinarlos eficientemente.

## Estrategias por Trabajo

### Minero 🪨

**Pagos destacados:**
- Diamante: **20.0** monedas
- Esmeralda: **25.0** monedas 
- Ancient Debris: **50.0** monedas

**Estrategia eficiente:**
1. Mina en el nivel Y=-58 para encontrar diamantes (mayor concentración)
2. Usa encantamiento Fortuna III para multiplicar tus ganancias
3. Busca biomas de montaña para esmeraldas
4. Explora el Nether a nivel Y=15 para Ancient Debris
5. Utiliza el método de minado en rama para maximizar eficiencia

### Leñador 🌳

**Pagos:**
- Troncos normales: **2.5** monedas por bloque
- Troncos descortezados: **1.5** monedas

**Estrategia eficiente:**
1. Tala árboles grandes como robles oscuros y abetos gigantes
2. Usa hachas con Eficiencia V y Durabilidad
3. Planta tus propios bosques organizados en filas para más eficiencia
4. Evita descortezar los troncos, ya que pagan menos

### Granjero 🌾

**Pagos destacados:**
- Trigo y remolacha madura: **2.5** monedas
- Criar animales: **5.0-5.5** monedas
- Domesticar animales: **7.0-8.0** monedas

**Estrategia eficiente:**
1. Crea granjas automáticas de caña de azúcar, melones y calabazas
2. Establece granjas de cría con dispensadores de agua para facilitar recolección
3. Domestica caballos y lobos por las mayores recompensas
4. Esquila ovejas regularmente (regeneran su lana rápidamente)
5. Aprovecha el sistema de pago por bloques apilados para caña de azúcar

### Pescador 🎣

**Pagos destacados:**
- Bacalao: **5.0** monedas
- Salmón: **6.0** monedas
- Pez tropical/globo: **8.0** monedas

**Estrategia eficiente:**
1. Usa cañas encantadas con Suerte del Mar III y Atrayente III
2. Pesca en aguas abiertas (no en estanques artificiales pequeños)
3. Combina con AFK en momentos de descanso
4. Pesca durante días de lluvia para mejorar resultados

### Constructor 🏗️

**Pagos destacados:**
- Bloques de madera: **2.5** monedas
- Escaleras especiales: **3.0** monedas
- Cofres, mesas de encantamiento: **3.5** monedas
- Obsidiana: **4.0** monedas

**Estrategia eficiente:**
1. Construye con bloques que pagan más (escaleras, cofres, librerías)
2. Usa dispensadores para colocar bloques más rápido
3. Prioriza estructuras funcionales sobre decorativas
4. Construye en patrones repetitivos para mayor eficiencia

### Cazador 🏹

**Pagos destacados:**
- Mobs hostiles comunes: **4.0-4.5** monedas
- Enderman: **6.0** monedas
- Wither: **100.0** monedas
- Dragón: **500.0** monedas

**Estrategia eficiente:**
1. Construye granjas de mobs (con spawners o naturales)
2. Busca biomas donde spawneen endermans con frecuencia
3. Usa espadas con Filo y Saqueo para maximizar ganancias
4. Organiza grupos para cazar jefes como el Wither regularmente

### Alquimista 🧪

**Pagos destacados:**
- Ingredientes básicos: **5.0-6.0** monedas
- Ingredientes intermedios: **8.0-10.0** monedas
- Aliento de dragón: **20.0** monedas

**Estrategia eficiente:**
1. Automatiza la producción de ingredientes comunes
2. Cultiva verrugas del nether para la base de pociones
3. Crea granjas de brujas para obtener redstone, glowstone y azúcar
4. Almacena aliento de dragón cuando enfrentes al dragón

## Zonas Especiales para Farmeo

### Las Mejores Ubicaciones para cada Trabajo

1. **Minería:**
   - El bioma de Badlands para oro
   - Cuevas profundas (-59) para diamantes
   - Ancient Debris: coordenadas Y=15 en el Nether

2. **Agricultura:**
   - Biomas de llanuras para espacio de cultivo
   - Establece tu base cerca de un cuerpo de agua para irrigación

3. **Caza:**
   - Desiertos durante la noche para arañas y esqueletos
   - Pantanos para slimes
   - End externo para endermans

## Combinaciones de Trabajos Óptimas

Para maximizar tu ganancia por hora, estas son las mejores combinaciones:

### Combinación de Alta Ganancia: Minero + Cazador + Alquimista
- **Ventaja:** Aprovecha los materiales más valiosos del juego
- **Implementación:** Mientras minas, también eliminas mobs (cazador) y obtienes materiales para pociones (alquimista)

### Combinación AFK: Pescador + Granjero
- **Ventaja:** Puedes alternar entre pesca AFK y recolección de cultivos automáticos
- **Implementación:** Establece sistemas semiautomáticos para ambos trabajos

### Combinación para Construcción: Constructor + Leñador + Minero
- **Ventaja:** Obtienes y procesas tus propios materiales, ganando en cada paso
- **Implementación:** Recolecta, procesa y construye, ganando en cada fase

## Cálculo de Eficiencia por Hora

Con las configuraciones actuales del servidor, estas son las ganancias aproximadas por hora:

- **Minero (diamantes)**: ~1000-1500 monedas/hora
- **Leñador (robles oscuros)**: ~800-1200 monedas/hora
- **Granjero (trigo + animales)**: ~900-1300 monedas/hora
- **Pescador (optimizado)**: ~700-1000 monedas/hora
- **Constructor (estructuras complejas)**: ~600-900 monedas/hora
- **Cazador (granja de mobs)**: ~1200-1600 monedas/hora
- **Alquimista (ingredientes raros)**: ~800-1100 monedas/hora

## Herramientas Esenciales

Para maximizar tus ganancias, asegúrate de tener:

1. **Herramientas con Eficiencia V**
2. **Armas con Saqueo III**
3. **Caña de pescar con Suerte del Mar III**
4. **Items con encantamiento Fortuna III**
5. **Botas con Caída de Pluma IV** (para minería y construcción)
