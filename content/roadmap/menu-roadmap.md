## 📝 Descripción General
Sistema integral de menús que proporciona una interfaz unificada y funcional para diversas características del servidor.

## 🎯 Objetivos
- [ ] Implementar un sistema de menús coherente y funcional
- [ ] Mejorar la experiencia de usuario en todas las interacciones
- [ ] Optimizar la navegación entre diferentes secciones
- [ ] Asegurar la consistencia visual en todos los submenús
